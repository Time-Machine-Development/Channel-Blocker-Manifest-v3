FAQ

Why does this add-on require permissions?

Access to Your Data on www.youtube.com:

- This permission is necessary for functionalities such as removing users/channels you have blocked on YouTube™ and adding the 'x'-button. It allows the add-on to interact with and modify your YouTube™ data as needed.

Access to Browser Tabs:

- This permission is essential for keeping track of your active YouTube™ tabs and managing the Add-on setting page tab.

Storage of Unlimited Client-Side Data:

- This permission is required to store an unlimited amount of client-side data, enabling the add-on to effectively block any number of users or channels.

Troubleshooting Guide: Add-on Not Functioning

Check for Updates

Ensure that you have the latest version of Channel Blocker installed. If not, update to the newest version for optimal performance.

Mobile Compatibility

If you are using a phone, we are sorry but this add-on doesn't work on mobile browsers.

Reporting a Bug

If you have the latest version and still encounter issues, use our bug report feature. Visit the YouTube™ page where the problem occurs, right-click with your mouse, and select the "Report a bug on this page" option.

Bug Report Submission

Complete the required fields in the newly created bug report tab, save the report, and send it to yt-cleaner@protonmail.com. Including detailed information helps us address and resolve the issue promptly.
